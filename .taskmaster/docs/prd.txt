# Document-Driven PE Analysis System

## Project Overview

Build an intelligent document monitoring system that automatically generates updated company analysis whenever new financial documents, research, or due diligence materials are added to private company folders in Google Drive. This creates "living analysis" that stays current as new information becomes available.

## The Real Workflow (Corrected Understanding)

### Current PE Firm Process
1. **Private Company Folders**: Each target company has dedicated Google Drive folder
   - `/Private Companies/CompanyA/` contains all CompanyA materials
   - `/Private Companies/CompanyB/` contains all CompanyB materials
   - etc.

2. **Document Collection**: PE team continuously adds new documents
   - Financial statements (quarterly, annual)
   - Crunchbase data exports  
   - Management presentations
   - Industry research reports
   - Due diligence findings
   - Third-party analysis reports

3. **Manual Analysis**: Currently someone manually reviews new docs and updates analysis

### Automated Solution Architecture (Enhanced with State Management)

```
New Document Added to /CompanyA/ folder
    ↓
Google Drive Change Detection (Push Notifications)
    ↓  
n8n Triggered for CompanyA Specifically
    ↓
Collect ALL Documents in CompanyA Folder  
    ↓
Parse/Extract Data + Extract Financial Metrics
    ↓
Update Time-Series State Database (Google Sheets)
    ↓
Calculate Deltas & Historical Trends
    ↓
Run Complete AI Analysis Using All Available Data + Historical Context
    ↓
Generate Updated CompanyA Analysis Report with Delta Intelligence
    ↓
Trigger Smart Alerts for Significant Changes
    ↓
Distribute to Stakeholders + Archive Previous Version
```

## Time-Series State Management Architecture

### Overview
Implement an append-only time-series database using Google Sheets to track company financial metrics over time. This enables instant delta calculation and historical trend analysis, providing PE teams with immediate insight into metric changes like "Company revenue increased $10M since last month."

### State Database Design

**Core Principles:**
- **Append-Only Pattern**: Never modify existing records, only add new entries
- **Time-Series Structure**: Every metric entry includes timestamp and document lineage
- **Delta Intelligence**: Automatic calculation of changes between time periods
- **Historical Context**: Preserve complete audit trail for all metric evolution

### Metric Extraction Engine

**Financial Metric Recognition Patterns:**
```javascript
const metricPatterns = {
  revenue: [
    /revenue[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)\s*([kmb]illion?|[kmb]|million?)?/i,
    /total[_\s]?revenue[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i,
    /sales[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i
  ],
  
  valuation: [
    /valuation[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)\s*([kmb]illion?|[kmb]|million?)?/i,
    /valued[_\s]?at[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i,
    /enterprise[_\s]?value[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i
  ],
  
  arr: [
    /arr[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i,
    /annual[_\s]?recurring[_\s]?revenue[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i
  ],
  
  burn_rate: [
    /burn[_\s]?rate[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i,
    /monthly[_\s]?burn[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i,
    /cash[_\s]?burn[:\s]*\$?([0-9,]+(?:\.[0-9]+)?)/i
  ]
};

// Extract metrics from document text
function extractMetrics(documentText, documentSource) {
  const extractedMetrics = {};
  
  for (const [metricName, patterns] of Object.entries(metricPatterns)) {
    for (const pattern of patterns) {
      const match = documentText.match(pattern);
      if (match) {
        extractedMetrics[metricName] = {
          value: parseFloat(match[1].replace(/,/g, '')),
          unit: match[2] ? normalizeUnit(match[2]) : 'USD',
          confidence: calculateConfidence(match, pattern),
          source: documentSource,
          rawMatch: match[0]
        };
        break; // Use first successful match
      }
    }
  }
  
  return extractedMetrics;
}
```

### Delta Intelligence System

**Real-Time Delta Calculation:**
```javascript
// Function to calculate deltas and trends
async function calculateDeltas(companyName, newMetrics) {
  const results = {};
  
  for (const [metricName, metricData] of Object.entries(newMetrics)) {
    // Get previous value from state database
    const previousValue = await getLatestMetric(companyName, metricName);
    
    if (previousValue) {
      const delta = metricData.value - previousValue.value;
      const deltaPercent = (delta / previousValue.value) * 100;
      const timeDiff = new Date() - new Date(previousValue.timestamp);
      const daysDiff = timeDiff / (1000 * 60 * 60 * 24);
      
      results[metricName] = {
        current: metricData.value,
        previous: previousValue.value,
        delta: delta,
        deltaPercent: deltaPercent,
        timePeriod: `${Math.round(daysDiff)} days`,
        trend: delta > 0 ? 'increasing' : delta < 0 ? 'decreasing' : 'stable',
        significance: Math.abs(deltaPercent) > 10 ? 'high' : 
                     Math.abs(deltaPercent) > 5 ? 'medium' : 'low'
      };
      
      // Trigger alerts for significant changes
      if (Math.abs(delta) > getAlertThreshold(companyName, metricName)) {
        await triggerDeltaAlert(companyName, metricName, results[metricName]);
      }
    } else {
      // First time seeing this metric
      results[metricName] = {
        current: metricData.value,
        previous: null,
        delta: null,
        deltaPercent: null,
        trend: 'new',
        significance: 'new'
      };
    }
  }
  
  return results;
}

// Smart alert system
async function triggerDeltaAlert(companyName, metricName, deltaData) {
  const alertMessage = `🚨 ${companyName} Alert: ${metricName} changed by $${Math.abs(deltaData.delta).toLocaleString()} (${deltaData.deltaPercent.toFixed(1)}%) in ${deltaData.timePeriod}`;
  
  const recipients = getDeltaAlertRecipients(deltaData.significance);
  
  await sendNotification({
    type: 'delta_alert',
    company: companyName,
    metric: metricName,
    message: alertMessage,
    recipients: recipients,
    data: deltaData
  });
}
```

**Step 3: Document Data Extraction + Metric Extraction**
```javascript
// Function Node - Extract Data + Financial Metrics from All Document Types

const extractors = {
  // PDF documents (financial reports, research)
  pdf: async (fileId) => {
    const file = await googleDrive.files.get({fileId, alt: 'media'});
    return await extractTextFromPDF(file.data);
  },
  
  // Excel/CSV financials  
  xlsx: async (fileId) => {
    const file = await googleDrive.files.get({fileId, alt: 'media'});
    const workbook = XLSX.read(file.data);
    return extractFinancialData(workbook);
  },
  
  // Word documents (notes, analysis)
  docx: async (fileId) => {
    const file = await googleDrive.files.get({fileId, alt: 'media'});
    return await extractTextFromDOCX(file.data);
  }
};

// Process all documents and extract relevant data + metrics
const extractedData = {};
const extractedMetrics = {};

for (const [category, files] of Object.entries($json.documents)) {
  extractedData[category] = [];
  
  for (const file of files) {
    const extractor = extractors[getFileExtension(file.name)];
    if (extractor) {
      const data = await extractor(file.id);
      
      // Extract financial metrics from document content
      const metrics = extractMetrics(data, file.name);
      
      extractedData[category].push({
        fileName: file.name,
        extractedData: data,
        lastModified: file.modifiedTime,
        extractedMetrics: metrics
      });
      
      // Accumulate metrics for state database update
      Object.assign(extractedMetrics, metrics);
    }
  }
}

return [{
  json: {
    companyName: $json.companyName,
    extractedData: extractedData,
    extractedMetrics: extractedMetrics,
    documentSummary: generateDocumentSummary(extractedData),
    metricsCount: Object.keys(extractedMetrics).length
  }
}];
```

**Step 4: State Database Update + Delta Calculation**
```javascript
// Function Node - Update Time-Series State + Calculate Deltas

const companyName = $json.companyName;
const newMetrics = $json.extractedMetrics;
const triggerFile = $json.triggerFile.fileName;

// Update state database (Google Sheets) with new metrics
for (const [metricName, metricData] of Object.entries(newMetrics)) {

    timestamp: new Date().toISOString(),

    metric_name: metricName,
    metric_value: metricData.value,
    unit: metricData.unit,
    confidence: metricData.confidence,
    extracted_by: 'n8n_pipeline',
    raw_match: metricData.rawMatch
  };
  
  // Append to appropriate Google Sheets
  await appendToStateDatabase(companyName, metricName, stateEntry);
}

// Calculate deltas against historical data
const deltaAnalysis = await calculateDeltas(companyName, newMetrics);

// Get historical context for AI agents
const historicalContext = await getHistoricalContext(companyName);

return [{
  json: {
    companyName: companyName,
    extractedData: $json.extractedData,
    extractedMetrics: newMetrics,
    deltaAnalysis: deltaAnalysis,
    historicalContext: historicalContext,
    stateUpdateComplete: true,
    significantChanges: Object.values(deltaAnalysis).filter(d => d.significance === 'high')
  }
}];
```

### Notification & Distribution System

**Smart Notifications Based on Changes + Delta Intelligence**
```javascript
// Function Node - Determine Notification Priority with Delta Awareness
const triggerFile = $json.triggerFile.fileName.toLowerCase();
const significantChanges = $json.significantChanges || [];
const deltaAnalysis = $json.deltaAnalysis || {};

let priority = 'normal';
let notificationGroup = ['team@pe-firm.com'];

// DELTA-DRIVEN PRIORITY (highest priority)
if (significantChanges.length > 0) {
  priority = 'critical';
  notificationGroup = ['partners@pe-firm.com', 'analysts@pe-firm.com', 'leadership@pe-firm.com'];
  
  // Check for extreme changes
  const extremeChanges = significantChanges.filter(change => 
    Math.abs(change.deltaPercent) > 25 || Math.abs(change.delta) > 5000000
  );
  
  if (extremeChanges.length > 0) {
    priority = 'emergency';
  }
}

// DOCUMENT-BASED PRIORITY (secondary)
else if (triggerFile.includes('financial') || triggerFile.includes('q1') || triggerFile.includes('q2') || 
         triggerFile.includes('q3') || triggerFile.includes('q4') || triggerFile.includes('annual')) {
  priority = 'urgent';
  notificationGroup = ['partners@pe-firm.com', 'analysts@pe-firm.com'];
}

// Medium priority triggers  
else if (triggerFile.includes('research') || triggerFile.includes('market') || 
         triggerFile.includes('competitive') || triggerFile.includes('crunchbase')) {
  priority = 'high';
  notificationGroup = ['team@pe-firm.com', 'research@pe-firm.com'];
}

// Generate delta summary for notification
const deltasSummary = Object.entries(deltaAnalysis)
  .filter(([_, data]) => data.significance === 'high')
  .map(([metric, data]) => `${metric}: ${data.delta > 0 ? '+' : ''}$${Math.abs(data.delta).toLocaleString()} (${data.deltaPercent.toFixed(1)}%)`)
  .join(', ');

const subject = significantChanges.length > 0 
  ? `${priority.toUpperCase()}: ${$json.companyName} - SIGNIFICANT METRICS CHANGE: ${deltasSummary}`
  : `${priority.toUpperCase()}: ${$json.companyName} Analysis Updated - New ${triggerFile}`;

return [{
  json: {
    priority: priority,
    recipients: notificationGroup,
    subject: subject,
    companyName: $json.companyName,
    deltasSummary: deltasSummary,
    significantChangesCount: significantChanges.length,
    alertType: significantChanges.length > 0 ? 'delta_alert' : 'document_update'
  }
}];
```
